📖 Step-by-Step: Customer Payment Guide

Ye steps aap customer ko share kar sakte hain order ke baad:

01

Binance Account Banao / Login Karo

Agar account nahi hai to binance.com ya Binance app se sign up karo. KYC verify karo (identity verification). Pakistan mein Binance fully accessible hai.

02

USDT Purchase Karo

Binance P2P section mein jao → Buy USDT select karo → Pakistani Rupees (PKR) mein payment karo EasyPaisa / JazzCash / Bank Transfer ke zariye. Amount approximately calculate karo: 1 USDT ≈ $1.

03

AURA STORE Wallet Address Copy Karo

Order place karne ke baad hum tumhe USDT wallet address bhejenge. Ya upar diya gaya address use karo. Yeh TRC20 network use karna — transaction fee sabse kam hoti hai.

04

Send Karo

Binance Wallet → Withdraw → USDT select karo → address paste karo → Network TRC20 select karo → Amount enter karo → Confirm. Transaction 1-5 minute mein complete hoti hai.

05

Screenshot Share Karo

Transaction successful hone ke baad Transaction ID (TXID) ya screenshot humein WhatsApp / Discord pe bhejo. Hum confirm karke account instantly deliver karein ge.

💡

Pro Tip: TRC20 network use karo — transaction fee sirf $1 ya $2 hoti hai. ERC20 bohot zyada fee leta hai. Minimum send amount usually 1 USDT hota hai Binance pe. Customers ko guide karo ke network galat select na ho — wrong network pe send kiya gaya USDT recover nahi hota.
